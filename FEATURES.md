## 3. 메인 페이지 기능 (index.html + room3d.js)

### 3.1 3D 환경

- **공간 구성**: 바닥(20x20), 벽 3개, 천장
- **조명**: 천장 매달린 조명 12개 (원형 배치, 포인트 라이트)
- **그림자**: Shadow mapping 활성화
- **안개 효과**: 15~40 거리

### 3.2 캐릭터 (White Hat Hacker)

**구조:**
- 마인크래프트 스타일 복셀 캐릭터
- 머리, 몸통, 팔 2개, 다리 2개 (관절 포함)
- 화이트햇 모자 (화이트해커 상징)
- 얼굴: ^ ^ 눈, 미소 입
- 오른손 리모컨 (사이버펑크 스타일)

**애니메이션:**
| 동작 | 설명 |
|------|------|
| 걷기 | 다리/팔 자연스러운 스윙 |
| 앉기 | 쇼파에 앉는 모션 |
| 일어서기 | 앉기 역재생 |
| 오락기 플레이 | 팔 움직임 |
| 트레드밀 달리기 | 다리 운동 |
| 집사 인사 | 허리 굽혀 인사 |

### 3.3 상호작용 오브젝트

| 오브젝트 | 기능 |
|----------|------|
| 작업 데스크 | 모니터, 키보드, 마우스 포함 |
| 오락기 | 게임 플레이 애니메이션 |
| 피아노 | 장식 |
| 운동 기구 | 트레드밀 달리기 |
| 쇼파 | 앉기 가능 |
| 해킹 터미널 | 검은 모니터 |
| 홀로그램 프로젝터 | 장식 |
| 서버 랙 | 장식 |
| 자격증 전시대 | 자격증 표시 |
| CTF 트로피 선반 | CTF 수상 표시 |
| 경력 TV 화면 | 경력 표시 |
| 수상 트로피 선반 | 수상이력 표시 |

### 3.4 조작법

| 입력 | 기능 |
|------|------|
| **좌클릭** | 캐릭터 이동 |
| **우클릭 드래그** | 카메라 회전 |
| **마우스 휠** | 줌 인/아웃 (5~30) |
| **E키** | 오브젝트 상호작용 |
| **F키** | 쇼파 앉기/일어나기 |
| **Escape** | 다이얼로그 닫기 |

### 3.5 UI 시스템

- **조작법 안내**: 좌상단 고정
- **상호작용 프롬프트**: 하단 중앙 (E키 안내)
- **다이얼로그**: 모달 팝업
- **CMD 윈도우**: 터미널 스타일 프로필 (수상이력/자격증/경력 탭)
